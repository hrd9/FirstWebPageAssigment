IS601 Assignment 2 : Git Command Line and First Website Creation

The different types of commands that we have studied in class are divided mostly into three categories. They are as follows :

  • Linux Commands

    1. ls : It is use to list the directory of the file
    2. rm file : This command is use to delete the specific file
    3. mkdir : It is use to make the new directory in the workspace
    4. cp file1 file2 : It is use to copy the content of file1 to file2
    5. date : This is the command that is use to show the current date and time
    6. pwd : This command is use to illustrate the current directory of the work base

  • Git Commands

    1. git init : It is use to initialize an existing working directory as a Git Repository
    2. git branch : This command is use to create a new branch at the current commit
    3. git checkout : It is use to switch to another branch and check it out into your working directory
    4. git status : It shows the modified file in the working directory and staged it for your next commit
    5. git commit -m : This command is use for commit your stage content as a new commit snapshot

  • Vi Editor Commands

    1. vi filename : This command is use to edit the filename starting at line 1
    2. wq : It is use to quit vi mode, writing out modified file to file named in original invocation
    3. q : In order to exit the from the vi mode, quit command is use
    4. i : This command is use to insert text before cursor, until we press the ESC key
    5. (Colon) w : It is use to write the current contents to file named in original vi call
